# Study Hub (স্টাডি হাব) - class 9, 10 learning application

স্টাডি হাব হলো ৯ম ও ১০ম শ্রেণির শিক্ষার্থীদের জন্য একটি আধুনিক, ডাইনামিক এবং অত্যন্ত আকর্ষণীয় ওয়েব অ্যাপ্লিকেশন। এই প্ল্যাটফর্মে শিক্ষার্থীরা বিভিন্ন বিষয়ের স্মার্ট নোট পড়তে পারবে, ভিডিও ক্লাস দেখতে পারবে এবং MCQ পরীক্ষায় অংশ নিয়ে নিজের মেধা যাচাই করতে পারবে। অভিভাবক এবং অ্যাডমিনদের জন্য এতে রয়েছে আলাদা ড্যাশবোর্ড সুবিধা।

## 🌟 মূল ফিচারসমূহ (Features)

*   **স্টুডেন্ট ড্যাশবোর্ড (Student Dashboard):**
    *   **স্টাডি স্ট্রিক ক্যালেন্ডার (Study Streak):** প্রতিদিনের পড়ার ধারাবাহিকতা ধরে রাখার জন্য গিটহাব-স্টাইল হিটম্যাপ।
    *   **অ্যানালিটিক্স (Study Analytics):** সাপ্তাহিকভাবে কোন বিষয়ে কত সময় পড়া হয়েছে তার ভিজ্যুয়াল রিপ্রেজেন্টেশন।
    *   **আসন্ন ডেডলাইন:** পরীক্ষা এবং অ্যাসাইনমেন্টের সময়সীমা সম্পর্কিত রিমাইন্ডার।
*   **অনলাইন MCQ পরীক্ষা:**
    *   টাইমার সহ লাইভ MCQ পরীক্ষা দেওয়ার সুবিধা।
    *   পরীক্ষা শেষে ফলাফল বিশ্লেষণ, ভুল উত্তরের সঠিক ব্যাখ্যা এবং ট্রফি অ্যানিমেশন।
*   **স্মার্ট নোট ও লেকচার শিট:**
    *   স্ক্রল-ভিত্তিক রিডিং প্রগ্রেস ট্র্যাকার।
    *   ফন্ট সাইজ নিয়ন্ত্রণ এবং অফলাইন বুকমার্ক করার সুবিধা।
*   **এআই টিউটর অ্যাসিস্ট্যান্ট (AI Tutor):**
    *   যেকোনো বিষয়ে পড়ার যেকোনো জিজ্ঞাসায় সাহায্য করার জন্য চ্যাটবট (OpenAI API দ্বারা চালিত)।
*   **অভিভাবক ড্যাশবোর্ড (Guardian Dashboard):**
    *   সন্তানদের পড়াশোনার অগ্রগতি মনিটর করা।
    *   পেমেন্ট হিস্ট্রি ট্র্যাকিং এবং স্কুল ক্যালেন্ডার।
*   **অ্যাডমিন প্যানেল (Admin Dashboard):**
    *   সিস্টেমের স্বাস্থ্য (System Health), ব্যবহারকারী ও কনটেন্ট ম্যানেজমেন্ট।

## 🛠 প্রযুক্তিগত কাঠামো (Tech Stack)

*   **ফ্রন্টএন্ড:** Next.js 16 (App Router), React 19, TypeScript
*   **স্টাইলিং:** Tailwind CSS v4, Framer Motion (অ্যানিমেশন), Shadcn UI
*   **ডাটাবেজ ও ওআরএম:** SQLite Database, Prisma ORM
*   **এআই ইন্টিগ্রেশন:** OpenAI API

## 🚀 কিভাবে লোকাল মেশিনে রান করবেন (Installation & Setup)

১. **ডিপেন্ডেন্সি ইনস্টল করুন:**
   ```bash
   npm install
   ```

২. **পরিবেশ ভেরিয়েবল কনফিগার করুন:**
   রুট ডিরেক্টরিতে একটি `.env` ফাইল তৈরি করুন (অথবা বিদ্যমান `.env` ফাইলটি দেখুন) এবং নিচের ভেরিয়েবলগুলো যুক্ত করুন:
   ```env
   DATABASE_URL="file:./db/custom.db"
   OPENAI_API_KEY="your-openai-api-key"
   ```

৩. **ডাটাবেজ সেটআপ:**
   প্রিজমা ক্লায়েন্ট জেনারেট করতে রান করুন:
   ```bash
   npx prisma generate
   ```

৪. **ডেভেলপমেন্ট সার্ভার চালু করুন:**
   ```bash
   npm run dev
   ```
   সার্ভারটি চালু হলে ব্রাউজারে `http://localhost:3000` ঠিকানায় প্রজেক্টটি দেখতে পাবেন।
